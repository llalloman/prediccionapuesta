# Reglas de negocio

## Probabilidad implicita

```text
probabilidad_implicita = 1 / cuota_decimal
```

## Margen de la casa

```text
margen = suma(1 / cuota_resultado) - 1
```

## Edge

```text
edge = probabilidad_modelo - probabilidad_casa
```

## Valor esperado

```text
EV = P(modelo) * ganancia_si_gana - P(pierde) * monto_apostado
```

## Gestion de banca

No se recomienda superar el 5% de la banca en una sola apuesta.
